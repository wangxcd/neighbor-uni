# neighbor-uni
# 你居吾邻小程序项目介绍

## 一、项目概述

"你居吾邻"是一款基于UniApp框架开发的社区生活服务小程序，旨在构建邻里互助、信息共享的社区生态平台。通过整合社区资源，连接居民需求，打造便捷、温馨的社区生活体验。

## 二、主要功能点

### 2.1 社区信息发布
- **多种信息类型**：支持发布普通信息、二手交易、寻物启事、社区活动等
- **服务类信息**：代取代送、代购代办、代丢垃圾、陪玩陪练等便民服务
- **微信群招募**：方便社区居民建立兴趣群组
- **社区开店**：支持小型商户入驻

### 2.2 邻里互动系统
- **评论互动**：支持对信息进行评论和回复
- **点赞功能**：对优质内容进行点赞支持
- **分享传播**：一键分享到微信好友或朋友圈

### 2.3 生活区管理
- **常居生活区**：设置主要居住的生活区
- **关注生活区**：关注多个感兴趣的生活区
- **智能定位**：自动获取周边生活区信息

### 2.4 便民服务模块
- **小店/工作室**：展示社区周边商家信息
- **物业对接**：小区物业信息查询与登记
- **消费券系统**：支持消费券的购买、使用和管理

### 2.5 个人中心
- **账户管理**：余额查询、充值、提现功能
- **发布管理**：管理个人发布的信息
- **参与记录**：查看参与过的社区活动
- **投诉建议**：反馈问题和建议

## 五、安装教程

### 5.1 环境准备

1. **安装Node.js**：从官网下载并安装Node.js v14.x或更高版本
2. **安装HBuilderX**：下载并安装HBuilderX开发工具
3. **安装微信开发者工具**：用于小程序预览和调试

### 5.2 项目安装

```bash
# 克隆项目代码
git clone https://github.com/wangxcd/neighbor-uni.git

# 进入项目目录
cd neighbor-uni

# 安装依赖
npm install
```

### 5.3 配置说明

**配置文件位置**：`src/config/config.js`

```javascript
// 服务器配置
export default {
  baseUrl: 'https://api.example.com',  // API服务器地址
  timeout: 30000,                      // 请求超时时间
  version: '1.0.0'                     // 版本号
}
```

**微信小程序配置**：`src/manifest.json`

```json
{
  "name": "你居吾邻",
  "appid": "*****",
  "description": "社区生活服务小程序",
  "versionName": "1.0.0",
  "versionCode": "100"
}
```

### 5.4 运行项目

1. 打开HBuilderX，导入项目
2. 选择运行方式：
   - **微信小程序**：运行 → 运行到小程序模拟器 → 微信开发者工具
   - **H5**：运行 → 运行到浏览器
3. 在微信开发者工具中预览和调试

### 5.5 构建发布

```bash
# 构建微信小程序版本
npm run build:mp-weixin

# 构建H5版本
npm run build:h5

# 构建App版本
npm run build:app-plus
```
